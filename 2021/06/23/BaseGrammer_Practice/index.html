홍길동 876504-1234567

생년월일을 추출하여 결과값으로

# 홍길동님의 생일은

hong = ‘8765041234567’
print(hong[:2],”년”,hong[2:4],”월”,hong[4:6],”일”)

if(int(hong[6]) % 2 == 1):
print(“남자”)
else:
print(“여자”)
```
