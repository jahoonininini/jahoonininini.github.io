기본 배열을 만드는 방법

NumPy 배열을 만들려면 함수를 사용할 수 있다. np.array()

요소 추가, 제거 및 정렬

  • 요소 정렬 - np.sort()

배열의 모양과 크기

  • ndarray.ndim - 축 또는 치수 수
  • ndarray.size - 배열의 총 요소 수
  • ndarray.shape - 배열의 각 치수에 저장된 요소 수를 나타내는 정수 튜플

배열 바꾸기

  • arr.reshape()
  • newshape 은 원하는 모양의 배열로 만들어 준다.
  • order:C C와 같은 인덱스 순서를 사용하여 요소를 읽고 쓰는 것을 의미

1D 배열을 2D 배열로 변환

  • np.newaxis
    배열의 치수가 한 차원씩 증가한다.
    1D -> 2D, 2D -> 3D
  • np.expand_dims

인덱싱 및 슬라이싱

Python 목록을 분할하는 것과 동일한 방법으로 Numpy 배열을 인덱싱하고 분할할 수 있다.

5 보다 작은 배열의 모든 값을 쉽게 인쇄 할 수 있다.

5 보다 크거나 같은 숫자를 선택하고 해당 조건을 사용하여 배열을 인덱싱 할 수도 있다.

2 로 나눌 수 있는 값을 인덱싱할 수도 있다.

$ five_up = (a > 5) | (a == 5)
$ print(five_up)
[[False False False False]
[ True True True True]
[ True True True True]]

np.nonzero()

5 미만의 값을 출력할 수 있다.

찾고 있는 값이 배열에 존재하지 않으면 반환된 인덱스 배열이 비어 있다.

vsstack 을 사용하여 세로로 쌓을 수 있다.

또는 hstack 을 사용해 가로로 쌓을 수 있다.

hsplit 을 통해 배열을 작은 배열로 분할 할 수 있다.
