기본 배열을 만드는 방법

NumPy 배열을 만들려면 함수를 사용할 수 있다. np.array()

요소 추가, 제거 및 정렬

  • 요소 정렬 - np.sort()

배열의 모양과 크기

  • ndarray.ndim - 축 또는 치수 수
  • ndarray.size - 배열의 총 요소 수
  • ndarray.shape - 배열의 각 치수에 저장된 요소 수를 나타내는 정수 튜플

배열 바꾸기

  • arr.reshape()
  • newshape 은 원하는 모양의 배열로 만들어 준다.
  • order:C C와 같은 인덱스 순서를 사용하여 요소를 읽고 쓰는 것을 의미

1D 배열을 2D 배열로 변환

  • np.newaxis
    배열의 치수가 한 차원씩 증가한다.
    1D -> 2D, 2D -> 3D
  • np.expand_dims
    ```
    $ a = np.array([1, 2, 3, 4, 5, 6])
    $ print(a.shape)
    (6,)

$ b = np.expand_dims(a, axis=1)
$ b.shape
(6, 1)

$ c = np.expand_dims(a, axis=0)
$ c.shape
(1, 6)
